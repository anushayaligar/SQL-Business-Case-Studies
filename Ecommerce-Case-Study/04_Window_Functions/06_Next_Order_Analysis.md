# Next Order Analysis

## Business Problem

Identify the next order amount for each customer.

## Objective

Understand future purchase patterns and customer purchasing progression.

## SQL Concepts Used

* Window Functions
* LEAD()
* PARTITION BY
* ORDER BY
* CTE

## Tables Used

* orders

## Findings

1. Future order values help analyze customer purchase progression.
2. Patterns can reveal increasing or decreasing spending behavior.
3. Useful for forecasting customer value.

## Business Recommendations

1. Use purchasing trends for targeted marketing.
2. Identify customers likely to increase spending.
3. Improve customer retention strategies.

## Skills Demonstrated

* Customer Analytics
* Window Functions
* Predictive Analysis
* Data Analytics
