# Previous Order Comparison Analysis

## Business Problem

Compare each customer's current order amount with their previous order amount.

## Objective

Analyze customer purchasing behavior and identify spending patterns over time.

## SQL Concepts Used

* Window Functions
* LAG()
* PARTITION BY
* ORDER BY
* CTE

## Tables Used

* orders

## Findings

1. Customers may increase or decrease spending between purchases.
2. Previous order analysis helps understand customer buying behavior.
3. Significant decreases may indicate churn risk.

## Business Recommendations

1. Monitor customers with declining spending.
2. Offer targeted promotions to re-engage customers.
3. Identify high-growth customers for upselling opportunities.

## Skills Demonstrated

* Customer Behavior Analysis
* Window Functions
* Trend Analysis
* Data Analytics
